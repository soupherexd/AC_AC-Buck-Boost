# JIXUN1_SPWM — TI F28069 Off-Grid Inverter (SPWM)

## Project Identity

- **Part of ACAC** (appears in `main.c` header: `/* ACAC */`)
- Single-chip SPWM inverter control on **TMS320F28069** (C2000 Delfino)
- TI **Code Composer Studio 11.2** project, COFF ABI (not ELF), `--abi=coffabi`
- Compiler: TI C2000 v21.6.0.LTS (Debug), v22.6.1.LTS (Release)
- Debugger: **XDS100v2 USB** (connect on startup, JTAG via GPIO0)
- Project name: `JIXUN1_SPWM`

| Config | Linker File | Runs From | Preprocessor |
|--------|-------------|-----------|-------------|
| Debug  | `28069_RAM_lnk.cmd` | SARAM | *(none needed)* |

## Application Architecture

All code is in a single file: **`main.c`** (~1044 lines). No RTOS.

```
main loop ──→ OLED_output() (polled, non-blocking)
ADC ISR   ──→ all control logic (runs at PWM rate)
ePWM1 ISR ──→ declared but NOT used in main() (commented out)
```

### Execution Flow
```
InitSysCtrl → PIE → EPWM1/2/3_Init → InitADC → PID1/2_Init → OLED_Init → EINT → main loop
```
- ADC ISR (`adc_isr`) is the heart: samples voltages/currents, runs PID, adjusts PWM duty cycle
- Control loop runs from ADC interrupt (triggered by ePWM3 SOCA at TBPRD in up-down mode = **20kHz**)
- `Ts = 0.0002` used in PLL/SOGI (discrete-time integrator step)

### Three Operating Modes (`tag`)
| tag | Mode | Control |
|-----|------|---------|
| 0   | Shutdown | DUTY=0.4, all PWM off |
| 1   | Voltage regulation | RMS feedback via PID1, closed-loop voltage |
| 2   | Current regulation | Inductor current via PID2 + PLL synchronization |

Toggle via KEY3 cycles through `[0,1,0,1]`.

## Key Peripherals

### ePWM (20kHz switching)
- **TBPRD = 2250**, up-down count → 90MHz / (2 × 2250) = **20kHz**
- **EPWM1/EPWM2**: H-bridge complementary outputs with dead-band
  - `DBRED = DBFED = 10` → ~111ns dead time (10 × 1/90MHz)
- **EPWM3**: ADC trigger source (SOCA at CTR=TBPRD)
- Only EPWM1/2/3 inited; EPWM4-8 are commented out

### ADC (12-bit, 0-3.3V)
- 9 channels configured, triggered by ePWM3 SOCA
  - A0: U_in, A1: (reserved), B2: (reserved), B1: (reserved), A2: (reserved), A3: U_out, B0: I_L, B3: (reserved), A4: (reserved)
  - **Active**: A0 (U_in), A3 (U_out), B0 (I_L)
- Calibration constants in `adc_isr`: `(raw - offset)/scale`
  - U_in:  `(ADCRESULT0 - 2050.7) / 36.237`
  - U_out: `(ADCRESULT5 - 2053.7) / 36.758`
  - I_L:   `(ADCRESULT6 - 2087.9) / 325.51`

### OLED (I2C, GPIO bit-banged)
- **Not** using hardware I2C peripheral. Bit-banged on GPIO28(SDA)/GPIO29(SCL)
- Custom driver: `F2806x_common/source/iic.c` + `F2806x_common/source/oled.c`
- SPI-type display: `OLED_WriteCommand()` / `OLED_WriteData()`
- Shows: U_ref, Duty_Cycle, tag number on 4 lines

### Key Input (4 buttons, active-low with pull-ups)
| Key | GPIO | Function |
|-----|------|----------|
| KEY1 | GPIO56 | U_REF += 0.5 (tag=1) / I_REF += 0.01 (tag=2) |
| KEY2 | GPIO54 | U_REF -= 0.5 (tag=1) / I_REF -= 0.01 (tag=2) |
| KEY3 | GPIO39 | Toggle mode (tag) |
| KEY4 | GPIO19 | *(init commented out, no action in code)* |

Key scan runs every ~8000 ADC ISR calls (~0.4s at 20kHz).

## SOGI-PLL (Software Phase-Locked Loop)
- Second-order generalized integrator for grid synchronization
- Tracks U_in angle via q-axis error → PI → frequency correction
- Limits: 85π to 115π rad/s (42.5–57.5 Hz)
- Output: `theta` (0–2π)

## PID Controllers
| PID | Input | Kp | Ki | Kd | Integral Clamp |
|-----|-------|----|----|----|----------------|
| PID1 (voltage) | U_RMS (RMS of U_out) | 0.00005 | 0 | 0 | ±15 |
| PID2 (current) | I_L (inductor current) | 0.00015 | 0 | 0 | ±20 |

Both use **P-only control** (Ki=Kd=0) with integral term clamped but unused (Ki=0) — the integral accumulator is maintained but the gain multiplier zeroes it. P-only with deadzone (±0.03 error threshold).

## Gotchas & Migration Notes

1. **Include paths in `.cproject`** contain stale absolute paths (`Star.Dust`, `Souphere` desktops). Preferred include: `${PROJECT_ROOT}` and `D:\work\CCS\TMS320F28069\F2806x_*\include`. Verify on new workspace.

2. **Output format is COFF** (`.out`), not ELF. Binary parser is CoffParser. The project cannot link ELF object files.

3. **Single `main.c`** — if splitting into multiple modules, update `sources.mk` (generated by CCS) or the project's source list in `.cproject`.

4. **No README existed** before this AGENTS.md.

5. **H-bridge polarity switching** in `adc_isr`:
   - Positive U_in: EPWM1A/1B active (complementary switching), EPWM2A/2B always on
   - Negative U_in: EPWM2A/2B active, EPWM1A/1B always on
   - Near zero: both pairs switch complementary

6. **Dead-time inserts** in `EPWM1-3_Init()` use `DBCTL.bit.POLSEL=2` (active high complementary — AHC mode) with `OUT_MODE=3` (both edges delayed). This is overwritten dynamically in `adc_isr` for polarity control.

7. **ADC calibration** — `AdcOffsetSelfCal()` is called in `InitADC()`. The scale/offset constants in `adc_isr` may need recalibration for each hardware unit.
